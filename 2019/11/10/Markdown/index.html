3. 超链接

Markdown 支持两种超链接语法：行内式和参考式，行内式用的比较多。

3.1 行内式

语法说明：

[link text](link address “link tittle”)

code:

The preview of the code above:

测试一下你被墙没有Google

测试一下你有木有网百度

3.2 参考式

参考式一般用于学术论文上，或者是一处链接在文章多处使用，方便统一管理。

语法说明：
参考式分为两部分，文中引用为[link text][link mark]，在文中任意位置添加[link mark]:link address “link tittle”，link address和link tittle之间有空格。

如果link text本身可以作为链接标记，你也可以写成[link text][]
[link text]:link address “link tittle”

code:

测试一下你被墙没有Google

测试一下你被墙没有Google

3.3自动链接

Markdown支持使用比较简短的自动链接形式来处理网址和电子邮件，只要用<>把链接包起来，Markdown就会自动把它转成链接地址。一般网址的链接文字就和链接地址一样了，例如：

Code:

The preview of the code above:

http://www.baidu.com

锚点

网页中，锚点就是页内的超链接，也就是文档内部的元素，实现当前页中的跳转。如此，在这添加一个锚点，跳转到目录。在目录中点击这一节，就跳转到对应的章节。还有下一节的注脚。这些都是用锚点来实现的。

注意：

  1. Markdown Extra只支持在标题后插入锚点，其他地方无效。
  2. leanote编辑器右侧显示效果区域暂时不支持锚点跳转，所以点来点去没有反应，不必惊慌。
    语法说明：

在你准备跳转到的指定标题后插入锚点，然后在文档其他地方写上连接到锚点的链接。
